## Lucide Icons

This project uses [Lucide](https://lucide.dev) for icons. Lucide is a community-driven fork of Feather Icons.

Example usage:

```tsx
import { Home, Search, Settings } from "lucide-react";

function MyComponent() {
    return (
        <div>
            <Home />
            <Search />
            <Settings />
        </div>
    );
}
```
