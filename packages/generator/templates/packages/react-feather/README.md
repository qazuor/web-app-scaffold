## Feather Icons

This project uses [Feather Icons](https://feathericons.com) for icons. Feather Icons provides a clean and consistent icon set.

Example usage:

```tsx
import { Home, Search, Settings } from "react-feather";

function MyComponent() {
    return (
        <div>
            <Home />
            <Search />
            <Settings />
        </div>
    );
}
```
