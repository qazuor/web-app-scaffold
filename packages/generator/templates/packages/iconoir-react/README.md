## Iconoir Icons

This project uses [Iconoir](https://iconoir.com) for icons. Iconoir provides a comprehensive set of open source icons.

Example usage:

```tsx
import { Home, Search, Settings } from "iconoir-react";

function MyComponent() {
    return (
        <div>
            <Home />
            <Search />
            <Settings />
        </div>
    );
}
```
